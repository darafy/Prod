# Crypto Token Analyzer

## Name
Crypto Token Analyzer

## Description
An AI-powered Web3 skill that analyzes cryptocurrency projects, tokenomics, holder distribution, community activity, and risk factors.

## Capabilities

- Token analysis
- Holder distribution review
- Tokenomics evaluation
- Community assessment
- Risk scoring
- Bullish and bearish insights

## Input

- Token name
- Token ticker
- Contract address
- Blockchain network

## Output

- Project summary
- Token utility analysis
- Holder analysis
- Risk assessment
- Growth opportunities
- Overall score (0-100)
